# OSSPath

OSSPath helps developers discover open-source repositories and issues that match their skills and contribution experience.

## Problem

Many developers want to contribute to open source but struggle to:

- Find suitable repositories
- Identify beginner-friendly issues
- Track contribution progress
- Prepare for programs like LFX and GSoC

OSSPath solves this by analyzing GitHub activity and recommending relevant contribution opportunities.

---

## Features

- GitHub OAuth Authentication
- Skill Extraction from GitHub Repositories
- Repository Recommendations
- Issue Recommendations
- Open Source Readiness Score
- Contribution Analytics
